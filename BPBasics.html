Better chat radios.

That system add's new commands and configurations to make better chats radios, this system is inspired in the radios system of RoleCommands Plugin, in the config file you can create custom private radios, set the public radios channels, change the radio chat format, set if the players needs a item to use the radio and what item need and more...


Private radio channels

In the private radio channel config you can add custom members or set the radio only for some jobs. The "name" field is the frequency.


Radio chat format

You can change the format as a radio message appears in the chat, use {0} in the string for player id, {1} for player username, {2} for frequency and {3} for message

Remove all crimes

This is such a simple system, just set the RomeveCrimes field in the Config.json file to true and the player will never have crimes again

Make npc's inmortals

In this system you need the map editor, go to the npc you want to set as immortal and in the data field add "Immortal", this disables the "hands up" action for that npc and other players cannot damage or handcuff it the npc

Disable crime experience lose

Si se establece como verdadero, no pierde la experiencia de su trabajo cuando comete un delito.
