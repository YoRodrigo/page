    Better chat radios.

    That system add's new commands and configurations to make better chats radios, this system is inspired in the radios system of RoleCommands Plugin, in the config file you can create custom private radios, set the public radios channels, change the radio chat format, set if the players needs an item to use the radio and what item need and more...


    Private radio channels

    In the private radio channel config you can add custom members or set the radio only for some jobs. The "name" field is the frequency.


    Radio chat format

    You can change the format as a radio message appears in the chat, use {0} for player id, {1} for player username, {2} for frequency and {3} for message

    Remove all crimes

    This is such a simple system, just set the RomeveCrimes field in the Config.json file to true and the player will never have crimes again

    Disable crime experience lose

    If set true, the player don't lose experience of his job when his commit a crime.
